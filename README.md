# LPMSolo-v1.0
LPMSolo - Local Password Manager SOLO is a secure password manager that combines both fun and useful features! For example: You can generate passwords and immediately save them to a file below, then view all your passwords in a convenient format. You can also use the Magic Text section to encrypt words—you can reverse a word or restore it to its original form!
